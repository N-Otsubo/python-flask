# 図書管理システム 案

図書情報を管理し、貸出、返却、通知、ファイル出力を行う

## 管理情報
書籍
- ID
- 書籍名
- 著者
- ISBN
- 画像URL
- 書籍説明
- 残り冊数

貸出情報
- 書籍ID
- 学生ID（学籍番号）
- 貸出日

学生情報
- ID（学籍番号）
- 名前
- パスワード（ハッシュ）
- ソルト

管理者情報
- ID
- パスワード（ハッシュ）
- ソルト


## 画面構成
**ユーザ画面**
検索画面
- 借りている本一覧（ログイン時
- ユーザ情報変更（ログイン時
- ログイン（未ログイン時
- 検索結果一覧
 - 書籍詳細
   - 貸出完了画面


**管理者画面**
ログイン画面
 - 管理者マイページ
   - 書籍管理画面
     - 追加・編集・削除
   - ユーザ管理画面
     - 削除
     - 未返却者一覧
       - Excel出力
